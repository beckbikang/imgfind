Dependencies

pHash library (0.9.0-2 or compatible) Boost library (1.42.0.1 or compatible) For compilation: Development headers for all the above libraries cmake (2.6 or compatible) A C++ compiler.

Install

1. Clone repository 2. Run cmake 3. Compile with your chosen build-system/compiler 4. Copy the imgfind executable to wherever your system stores application binaries.
